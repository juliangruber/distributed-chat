
# Distributed Chat

**Talk slides and example code**.

To run the code,

```bash
$ git clone ...
$ npm install
$ node server.js 5000 &
$ open http://localhost:5000
$ node server.js 7000 5000 &
$ open http://localhost:7000
```

## The Code

`server.js` is the server part

`script.js` is the browser part

`index.html` is the required html

## Slides

```bash
$ open slide-{0,1,2}.md
```
